This are the descriptions of a data file in EGOBOO
The data structs are given in XML

data.txt ==> data.xml
Enclosing tag <data>
XML-Tag Description
Real general data
omitted Object slot number ( 0 to 63, don't use twice )
<class> Object class name ( use underscore for spaces ) Proposal: Name of class in 'classes.xml'
<unilit> Object is lit uniformly ( TRUE or FALSE )
<ammo> Maximum ammo ( 0 for doesn't matter )
Starting ammo ( 0 for doesn't matter ) 		[start] [max]
<gender> MALE or FEMALE or OTHER
Object stats
Proposal: 'Move this to classes.xml'
<lifebarcol> Life point bar color ( 0 to 5, 1 is red )
<manabarcol> Mana point bar color ( 0 to 5, 5 is purple )
<life> Life points ( eg. 3-5 )
<lifeinc> Life points increase per level ( eg. 1-3 )
<mana> Mana points ( eg. 0-0 )
<manainc> Mana points increase per level ( eg. 1-1 )
<manareturn> Mana return rate ( eg. 0-1 )
<manareturninc> Mana return increase per level ( eg. 0-1 )
<manaflow> Mana flow rate ( eg. 0-1 )
<manaflowinc> Mana flow increase per level ( eg. 0-1 )
<strength> Strength ( eg. 3-18 )
<strengthinc> Strength increase per level ( eg. 0-1 )
<wisdom> Wisdom ( eg. 3-18 )
<wisdominc> Wisdom increase per level ( eg. 0-1 )
<intelligence> Intelligence ( eg. 3-18 )
<intelligenceinc> Intelligence increase per level ( eg. 0-1 )
<dexterity> Dexterity ( eg. 3-18 )
<dexterityinc> Dexterity increase per level ( eg. 0-1 )
<size> Size ( eg. 1.0 )
<sizeinc> Size increase per level ( eg. 0.05 )
More physical attributes
<shadow> Shadow size ( eg. 30, 0 is none )
<bumpsize> Bump size ( eg. 30 )
Bump height ( eg. 70 ) 		[size] [height]
<bumpdampen> Bump dampen ( 0.0 to 1.0, lower is massive )
<weight> Weight ( 0 to 255 )
<jumppower> Jump power ( eg. 14.0 )
<numjumps> Number of jumps ( eg. 1 )
<speed> Sneak speed ( 1 to 20, 1 is good )
Walk speed ( 1 to 20, 3 is good )
Run speed ( 1 to 20, 5 is good )
<flyheight> Fly to height ( 0 is none )
<flash> Flashing AND ( 1, 3, 7, 15... 255 is don't )
<trans> Transparency blending ( 255 is normal )
<lightblend> Light blending ( 255 is normal )
<transblend> Transfer blending to weapons ( TRUE or FALSE )
<sheen> Sheen ( 0 - 15, 0 is dull, 15 is shiny )
<phong> Phong mapping ( TRUE or FALSE )
<texmove> Texture X movement rate ( eg. 0.0 )
Texture Y movement rate ( eg. 0.0 ) 		[x] [y]
<small> Conform to hills like a chair ( TRUE or FALSE )
Invulnerability data
<decorate> Object is totally invincible ( TRUE or FALSE ) Proposal: Replace by class DECORATE
omitted NonIFrame Invulnerability facing ( 0 to 65535 )
omitted NonIFrame Invulnerability angle ( 0 to 32768 )
omitted IFrame Invulnerability facing ( 0 to 65535 )
omitted IFrame Invulnerability angle ( 0 to 32768 )
Skin defenses
These are packed together for each skin
For <defshift> only values > 1 are taken into account
For <inversion> only the first TRUE value is taken into account
<skin>
  <no>[0..3]</no>
  <defbase>[Base defense rating of skin ( eg. 2 5 10 17 )]</defbase>
  <defshift>[.... defense shift ( eg. 1 1 1 1 )]</defshift>
  <inversion>[.... inversion ( eg. F F F T )]</inversion>
  <accel>[Acceleration rate ( eg. 40 38 30 20 )]</accel>
  <name>[Skin Name 0 ( use underscores for spaces )]</name>
  <cost>[Skin Cost 0 ( 0 to 65535 )]</cost>
</skin>
Experience and level data
These are packed together per level
Proposal: Move it to 'class.xml'
<level>
  <no>[0..5]</no>
  <exp>[Experience for level 1 ( 200 ) ]</exp>
<level>
<expstart> Starting experience ( eg. 0-500 )
<expworth> Experience worth for this object ( 10 ) Experience gained by killing this one
<expexchange> Experience exchange for this object ( 0.001 )
<exprate> [secret] [quest] [useunknown] [killenemy] [killsleepenemy] [killhated] [teamkills] [talk]
IDSZ Identification tags ( [NONE] is valid )
<idszparent> Parent ID ( eg. [FROG] for Trogs )
<idsztype> Type ID ( eg. [TROG] for Trogs )
<idszskill> Skill ID ( eg. [WMAG] for wizard spells )
<idszspecial> Special ID ( eg. [SUM0] for animated bones )
<idszhate> Hate group ID ( eg. [GOBL] for dwarves )
<idszdefense> Vuilnerability ID ( eg. [SILV] for werewolves ) Negative Defense -100: 100% vulnerable / +100: Not affected
Item and damage flags
<isitem> Object is an item ( TRUE or FALSE )
<ismount> Object is a mount ( TRUE or FALSE )
<isstackable> Object is stackable ( TRUE or FALSE )
<isidentified> Object's name is known ( TRUE or FALSE )
<usageknown> Item of known usage ( TRUE or FALSE )
<carrytomodule> Item carries to next module ( TRUE or FALSE )
<needskill> Item requires skill ID to use ( TRUE or FALSE )
<isplatform> Object is a platform ( TRUE or FALSE)
<collectsmoney> Object collects money ( TRUE or FALSE )
<canopenthings> Object can open chests/doors ( TRUE or FALSE )
Other item and damage stuff
<itemdamage> DamageTarget type ( SLASH, CRUSH, POKE, HOLY, EVIL, FIRE, ICE, or ZAP )
<itemattackuse> Attack type if it's a weapon ( UNARMED, THRUST, SLICE, CHOP, BASH, LONGBOW, XBOW, FLINGED, PARRY, ZAP)
Particle attachments
<attachnum> Number of particles to attach ( 0 )
<attachreaffirm> Reaffirm attachments when damaged ( NONE, SLASH, CRUSH, POKE, HOLY, EVIL, FIRE, ICE, ZAP,)
<attachprt> Particle type ( 0 to 3 )
Character hands
<gripvalid> Left ( or only ) grip valid ( TRUE or FALSE )
Right grip valid ( TRUE or FALSE ) 		[grip1] [grip2]
Particle spawning on attack order ( for weapon characters )
<hasattackprt> Particle attached to weapon ( TRUE or FALSE )
<attackprt> Particle type ( 0 to 3 )
Particle spawning for GoPoof
<poofamount> Amount to spawn ( 0 to 10, 0 is NoPoof )
<pooffacing> Facing add for each one ( 0 to 65535 )
<poofprt> Particle type ( 0 to 3 )
Particle spawning for Blud ( If you want it, you put it in... )
<bludvalid> Blud valid ( TRUE or FALSE )
<bludprt> Particle type ( 0 to 3 )
Extra stuff I forgot
<waterwalk> Character can walk on water ( TRUE or FALSE )
<bounciness> Character bounciness ( 0.00 to 0.95 )
More stuff I forgot
<healvalue> Life healing when used ( 0.0 to 100.0 )
<manacost> Mana cost to use ( 0.0 to 100.0 )
<lifereturn> Life return rate, ( 0 to 255 )
<stopmask> Stoppedby mask ( 48 is the only one )
Skin Name 0 ( use underscores for spaces ) Added to <skin> tag <name>
Skin Cost 0 ( 0 to 65535 ) Added to <skin> tag <cost>
<strengthdampen> Strength dampen for weapons ( 0.0 to 1.0 )

<canbedazed> Can be dazed ( TRUE or FALSE )
<canbegrogged> Can be grogged ( TRUE or FALSE )
<permanentlifeadd> Permanent life add when used ( -256 to 256 )
<permanentmanaadd> Permanent mana add when used ( -256 to 256 )
<canseeinvisible> Can see invisible ( TRUE or FALSE Proposal: Addas SKILL_...
<cursechance> Chance of being cursed ( 0 to 100 )
<sounds> Sound of Footfall ( -1 for none )
Sound of Jump ( -1 for none ) 		[footfall] [jump]
