This are the descriptions for the different types of characters in EGOBOO
The data structs are given in XML
SPELL is to replace the 'ENCHANT.TXT'

SPELL
XML-Tag Description
<name> Name of spell
<distance> USE_MELEE: On Character in 'Attack-Radius'
USE_FIRED: Effect is calculated if target is hit by 'THROWN' Particle.
<effect>
<type> How the effect has to be handled:
EFFTYPE_NONE
EFFTYPE_ATTACK
EFFTYPE_HEAL
<which> EFFECT_... like items
<dice> DICE: [Number of throws], [Number of sides], [added value]
<duration> In seconds
E.g for EFFECT_POISON, EFFECT_STRENGTH, EFFECT_SPEED
<repeat> E.g for EFFECT_POISON, repeated this much times. 0: Forever
<start_script> Script to run, as the spell is executed
(E. g. generate PARTICLE(s) which apply the end script if the a target is hit).
<end_script> Script to run, if the spell is applied on target. (E.g. play a sound)
