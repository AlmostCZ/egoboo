ITEM
General Info:
If an item-name is not given, it is generated off the item <type>
Data-Struct <item>
<number> Number of item (for inventory)
<idname> Number of Name, if not identified
<unidname> Number of Name, if identified
<flags> ITEMFLAG_IDENTIFIED, ITEMFLAG_MAGIC and so on...
<ammo> Depending on <use>
USE_FIREDWEAPON (e.g. Bow, Gun) : Number of shots
USE_WAND: Number of spells that can be cast (after that it has no effect)
<iconno> Which icon to use for display.
It is translated in an own table which model to use for 3D-Display
<acadd> Adds this amount of points to the armor-class (defense)
<wear> If item can be 'weared', then it adds AC and cann addd additional defense
Where it can be weared:
WEAR_NONE
WEAR_HELMET
WEAR_NECKLACE
WEAR_FINGER
WEAR_WRIST
WEAR_BODY: Armor --> Additional defense points
WEAR_FEET
<weareff> Effect on character, if the item is weared: Additional defense
EFFECT_... and value
<handbits> In which hand it can be held: LEFT, RIGHT, BOTH
<type> Type of item:
TYPE_NONE
TYPE_HELMET
TYPE_WEAPON
TYPE_SHIELD
TYPE_SHOES
TYPE_STONE
TYPE_ARROW
TYPE_KEY
TYPE_POTION
TYPE_RING
TYPE_WAND
TYPE_TWOHANDWEAPON
TYPE_SLING
TYPE_BOW
TYPE_DART
TYPE_SPELLBOOK
TYPE_CLERICHOLYSYMBOL
TYPE_ARMOR
TYPE_ROBE
TYPE_BRACERS
TYPE_MAGESCROLL
TYPE_CLERICSCROLL
<cooldown> Time to wait unit it can be used again (for WEAPONS), in tenths of a seconds
<skill> Skill needed to use this item:
SKILL_NONE
SKILL_FIGHTER
SKILL_MAGE
SKILL_CLERIC
SKILL_...

Is checked against a characters skill
<itemuse> How the item is used (to know for player characters):
ITEMUSE_SHIELD
ITEMUSE_MELEEWEAPON
ITEMUSE_FIREDWEAPON
ITEMUSE_RANGEDWEAPON
ITEMUSE_KEYS
ITEMUSE_CLERICSYMBOL
ITEMUSE_SPELLBOOK
ITEMUSE_RATION
ITEMUSE_WRISTS
ITEMUSE_CLERICSCROLL
ITEMUSE_MAGESCROLL
ITEMUSE_ARMOR
ITEMUSE_NECKLACE
ITEMUSE_MISC
ITEMUSE_STONEITEM
ITEMUSE_LOCKPICKS
ITEMUSE_16
ITEMUSE_COIN
ITEMUSE_POTION
ITEMUSE_WAND
ITEMUSE_HORN
ITEMUSE_RING
<effuse> How the <effect> has to be applied:
EFFUSE_OTHER
EFFUSE_ATTACK
EFFUSE_HEAL
<effect> See this struct
<singleuse> TRUE: Consume item after use (e. g. POTIONS, FOOD)
<value> Additional info about item, depending on <use>
USE_SHIELD: Additional defense point
USE_MELEEWEAPON: Additional damage for attack (< 0: cursed)
USE_FIREDWEAPON (e.g. Bow): Additional damage for attack (< 0: cursed)
USE_RANGEDWEAPON (e. g. Darts): Additional damage for attack (< 0: cursed)
USE_KEY: Number of key (To check out with locks for fit)
USE_CLERICSYMBOL: None
USE_SPELLBOOK: None
USE_RATION: How many food points to add(< 0: Rotten food)
USE_WRISTS: Additional defense points
USE_CLERICSCROLL: Number of 'spell' to cast / learn
USE_MAGESCROLL: Number of 'spell' to cast / learn
USE_ARMOR: Additional defense points
USE_NECKLACE: None
USE_MISC: None
USE_STONEITEM: Which item (To check out if used for fit)
USE_LOCKPICKS: None
USE_MISC2: None
USE_COIN: None
USE_POTION: Number of use effect (EFFECT_HEAL, EFFECT_POISON, EFFECT_STRENGTH, EFFECT_SPEED, EFFECT_FOOD, EFFECT_INVISIBLE, EFFECT_CUREPOISON
USE_WAND: Number of 'spell' to cast
USE_HORN: Number of sound to play
USE_RING: Number of effect
