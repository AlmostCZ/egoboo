Proposal

This is a description of a particle, how to write it in XML
The data structs are given in XML

Particle
XML-Tag Description Type of value
<force> Force spawn ( TRUE or FALSE ) char
<imgtype> Image type ( SOLID, LIGHT, or TRANS ) char
<frames> Starting frame ( 0 to 255 )
Number of frames ( 1 to 16 ) 		int[2]
<framerate> Frame rate ( 0 to 1000 )
Frame rate randomness ( 0, 1, 3, 7, 15, 31, 63... ) 		short int[2]
<rotation> Rotation base ( 0 to 65535 )
Random rotation add ( 0, 1, 3, 7, 15, 31, 63... )
Rotation rate ( 0 to 1000 ) 		int[3]
<size> Starting size ( 0 to 65535 )
Size increase rate ( 0 to 1000 ) 		int[2]
<speedlimit> Speed limit ( 0.0 to 10.0 ) float
<facerate> Facing rate ( 0 to 1000 ) short int
<weight> For physics:
0: Not affected by any physics
> 0: Drops to bottom (-z), accelerated by gravity
< 0: Goes up (+z) e. g. for smoke 		NEW
Ending conditions
<endwater> Water ends particle ( TRUE or FALSE )
<endbump> Bump ends particle ( TRUE or FALSE )
<endground> End if hits ground ( TRUE or FALSE )
<endframe> End after last frame ( TRUE or FALSE )
<endtime> Time before end ( 1 to 1000, 0 )
Collision data
<dampen> Bounce dampen ( 0.0 to 0.9 )
Bump money ( 0 to 100 ) See <effect><use>
EFFTYPE_MISC
ACTION_DROPMONEY
<bumpsize> Bump size ( 0 to 60 )
Bump height ( 0 to 60 ) 		int[2]
Bump damage amount ( 0-0 to 0-16 to 16-16 ) See <effect>
<use>EFFTYPE_ATTACK
<type>EFFECT_NONE
<dice>
<dynalight> Does it act as a dynamic light? ( TRUE or FALSE )
Dynalight level ( around 0.7 )
Dynalight falloff ( around 1000 ) 		char, float, int
Initial spawning of this particle
<facing> Facing base ( 0 to 65535 )
Facing randomness ( 0, 1, 3, 7, 15, 31, 63... ) 		int[2]
<xyspacing> XY Spacing ( 0 to 100 )
XY Spacing randomness ( 0, 1, 3, 7, 15, 31, 63... ) 		int[2]
<zspacing> Z Spacing ( -100 to 100 )
Z Spacing randomness ( 0, 1, 3, 7, 15, 31, 63... )
<xyvel> XY Velocity ( 0 to 100 )
XY Velocity randomness ( 0, 1, 3, 7, 15, 31, 63... )
<zvel> Z Velocity ( -100 to 100 )
Z Velocity randomness ( 0, 1, 3, 7, 15, 31, 63... )
Continuous spawning of other particles
<spawntime> Spawn time ( 0 to 100 ) char
<spawn> Amount to spawn each time ( 0 to 10 )
Facing add for each one ( 0 to 65535 )
Spawn type ( 0 to 3 )
<spawnend> Amount to spawn at end ( 0 to 10 )
Facing add for each one ( 0 to 65535 )
Spawn type ( 0 to 3 )
<spawnbump> Amount to spawn on bump ( 0 to 10 )
Spawn type ( 0 to 3 )
Random stuff
Daze time on hit ( 0 to 250 ) See <effect>
Grog time on hit ( 0 to 250 ) See <effect>
Spawn enchant on hit ( TRUE or FALSE ) See <effect>
Cause roll on hit ( TRUE or FALSE ) See <effect>
Cause pancake on hit ( TRUE or FALSE ) See <effect>
<targetneeded> Need target to spawn ( TRUE or FALSE ) Proposal: Move to <spell>
<targetcaster> Target is caster ( TRUE or FALSE ) Proposal: Move to <spell>
<targetstart> Start on target ( TRUE or FALSE ) Proposal: Move to <spell>
<damagefriendly> Only damage friendly ( TRUE or FALSE )
<sound> Spawn sound ( -1 for none )
End sound ( -1 for none )
<friendlyfire> Friendly fire ( TRUE or FALSE )
<onlyhithate> Only hit hate group ( TRUE or FALSE )
<spawnnewtarget> New target on spawn ( TRUE or FALSE ) Proposal: Move to <spell>
<targetangle> Target angle ( 0 to 32768 )
<dohoming> Home in on target ( TRUE or FALSE )
<homing> Homing friction ( 0.0 to 1.0 )
Homing acceleration ( 0.0 to 1.0 )
<rotatetoface> Rotate to face direction ( TRUE or FALSE )
<respawnhit> Respawn character on hit ( TRUE or FALSE )
Mana drain on hit ( 0 to 50 ) See <effect>
Life drain on hit ( 0 to 50 ) See <effect>
