ITEM
General Info:
If an item-name is not given, it is generated off the item <type>
Data-Struct <item>
<number> Number of item (for inventory)
<idname> Number of Name, if not identified
<unidname> Number of Name, if identified
<ammo> Depending on <use>
USE_FIREDWEAPON (e.g. Bow, Gun) : Number of shots
USE_WAND: Number of spells that can be cast (after that it has no effect)
<iconno> Which icon to use for display.
It is translated in an own table which model to use for 3D-Display
<type> Type of item:
TYPE_NONE
TYPE_HELMET
TYPE_WEAPON
TYPE_SHIELD
TYPE_SHOES
TYPE_STONE
TYPE_ARROW
TYPE_KEY
TYPE_POTION
TYPE_RING
TYPE_WAND
TYPE_TWOHANDWEAPON
TYPE_SLING
TYPE_BOW
TYPE_DART
TYPE_SPELLBOOK
TYPE_CLERICHOLYSYMBOL
TYPE_ARMOR
TYPE_ROBE
TYPE_BRACERS
TYPE_MAGESCROLL
TYPE_CLERICSCROLL
<cooldown> Time to wait unit it can be used again (for WEAPONS), in tenths of a seconds
<skill> Skill needed to use this item:
SKILL_NONE
SKILL_FIGHTER
SKILL_MAGE
SKILL_CLERIC
SKILL_...

Is checked against a characters skill
<value> Additional info about item, depending on <use>
USE_SHIELD: Additional defense point
USE_MELEEWEAPON: Additional damage for attack (< 0: cursed)
USE_FIREDWEAPON (e.g. Bow): Additional damage for attack (< 0: cursed)
(e. g. Darts): Additional damage for attack (< 0: cursed)
USE_KEY: Number of key (To check out with locks for fit)
USE_CLERICSYMBOL: None
USE_SPELLBOOK: None
USE_RATION: How many food points to add(< 0: Rotten food)
USE_WRISTS: Additional defense points
USE_CLERICSCROLL: Number of 'spell' to cast / learn
USE_MAGESCROLL: Number of 'spell' to cast / learn
USE_ARMOR: Additional defense points (< 0: cursed)
USE_NECKLACE: Spell-No if magic
USE_MISC: None
USE_STONEITEM: Which item (To check out if used for fit)
USE_LOCKPICKS: None
USE_MISC2: None
USE_COIN: None
USE_POTION: Number of use effect additional (EFFECT_HEAL, EFFECT_POISON, EFFECT_STRENGTH, EFFECT_SPEED, EFFECT_FOOD, EFFECT_INVISIBLE, EFFECT_CUREPOISON
NEW:Additional value for potion effect
USE_WAND: Number of 'spell' to cast
USE_HORN: Number of sound to play
USE_RING: Number of effect
<invbits> Can be put at tis places in the inventory (Bit-Wise)
<handbits> In which hand it can be held (left, right)
<acadd> Adds this amount of points to the armor-class (defense)
For ITEMS, that are weared.
<profallowed> Allowed for this profession (character class):
Can only be used by this character class (e. g. Wand only by mages)
<eff_type> How the effect has to be handled:
EFFTYPE_NONE
EFFTYPE_ATTACK
EFFTYPE_HEAL
<effect> Additional attack type:
EFFTYPE_NONE
EFFTYPE_SLASH
EFFTYPE_CRUSH
EFFTYPE_POKE
EFFTYPE_HOLY
EFFTYPE_EVIL
EFFTYPE_FIRE
EFFTYPE_ICE
EFFTYPE_ZAP
EFFTYPE_POISON
EFFTYPE_STONE
EFFTYPE_PARALYZE
EFFTYPE_SPEED
EFFTYPE_STRENGTH
EFFTYPE_FOOD
EFFTYPE_INVISIBLE
EFFTYPE_...

Is checked against the characters 'resistance'
Resistances are in percent [-100% (max. vulnerability) .. +100% (Not affected)]
An 'EFFTYPE_ATTACK's DICE is multiplied / divided by the resistance value. 0: Normal attack
<dice_small> DICE: [Number of throws], [Number of sides], [added value]
Attack value against small monsters
<dice_big> Attack value against big monsters
<duration> In seconds
E.g for EFFECT_POISON, EFFECT_STRENGTH, EFFECT_SPEED
<repeat> E.g for EFFECT_POISON, repeated this much times. 0: Forever
<use> Describes for the code, how the item is used
There are the following uses:
USE_SHIELD
USE_MELEEWEAPON
USE_FIREDWEAPON (e.g. Bow)
USE_RANGEDWEAPON (e. g. Darts)
USE_KEY (used in maze for locks)
USE_CLERICSYMBOL (for healer / paladin)
USE_SPELLBOOK (for mage)
USE_RATION (can be eaten)
USE_WRISTS (may have an effect on the wearer)
USE_CLERICSCROLL (healer can cast spell / learn spell)
USE_MAGESCROLL (mage can cast spell / learn spell)
USE_ARMOR (may have an effect on the wearer)
USE_NECKLACE (may have an effect on the wearer)
USE_MISC (special items for use in game)
USE_STONEITEM (special items for use in game)
USE_LOCKPICKS
USE_MISC2 (special items for use in game)
USE_COIN
USE_POTION (can be drunk, may effect the user)
USE_WAND (can be used by a mage for casting spells)
USE_HORN (special items for use in game, generates a sound)
USE_RING (may have an effect on the wearer)
